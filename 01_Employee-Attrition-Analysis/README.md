# Employee Attrition Analysis using Microsoft Excel

## 📌 Project Overview

This project analyzes employee attrition data to identify the key factors influencing employee turnover. The dataset was cleaned, transformed, and analyzed using Microsoft Excel, followed by the creation of an interactive dashboard that provides actionable business insights.

---

## 🎯 Objectives

- Analyze employee attrition trends.
- Identify departments and employee groups with the highest attrition.
- Understand factors affecting employee retention.
- Build an interactive dashboard for HR decision-making.

---

## 📂 Dataset

- Source: IBM HR Analytics Employee Attrition Dataset (Kaggle)
- Format: CSV → Converted to Excel (.xlsx)

---

## 🛠️ Data Preparation

The following data preparation steps were performed:

- Converted the dataset from CSV to Excel format.
- Checked for missing values.
- Checked for duplicate records.
- Verified data types and formatting.
- Cleaned inconsistent values.
- Added calculated columns where required.
- Prepared the dataset for Pivot Tables and Dashboard creation.

---

## 📊 Data Analysis

The dataset was analyzed using:

- Pivot Tables
- Pivot Charts
- Slicers
- Excel formulas
- KPIs
- Interactive Dashboard

---

## 📈 Dashboard KPIs

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Monthly Income
- Average Years at Company

---

## 🔍 Key Insights

- Employees working overtime have a significantly higher attrition rate.
- Sales department experiences the highest employee turnover.
- Employees with lower job satisfaction are more likely to leave.
- Younger employees tend to resign more frequently.
- Promotion delays are associated with increased attrition.
